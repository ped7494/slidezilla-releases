# 🦖 Slidezilla — Downloads

Este repositório hospeda **apenas os instaladores** do Slidezilla (app desktop de apresentações HTML). O código-fonte vive em repositório privado.

## Instalar (Windows)

Baixe o `Slidezilla-Setup-x.y.z.exe` da [release mais recente](https://github.com/ped7494/slidezilla-releases/releases/latest) e execute. Instalação em um clique, por usuário, sem privilégios de administrador.

## Atualizações automáticas

Depois de instalado, o app se atualiza sozinho: verifica novas versões a cada 15 minutos, baixa em segundo plano e aplica ao reiniciar (ou no próximo fechamento do app).

> Os arquivos `latest.yml` e `*.blockmap` de cada release são consumidos pelo auto-updater — não os remova das releases.
